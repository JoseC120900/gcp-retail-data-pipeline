# gcp-retail-data-pipeline
1. Descripción del proyecto
Este proyecto implementa un pipeline de datos en GCP para una empresa retail, permitiendo procesar y modelar datos de ventas para análisis de negocio.
2. Arquitectura
Cloud Storage → BigQuery (raw → staging → mart)
3. Tecnologías
- GCP (BigQuery, Cloud Storage)
- SQL
- Mockaroo (generación de datos)
4. Estructura del proyecto
/data
/sql
/scripts
/docs
5. Las reglas de calidad fueron definidas previamente en un catálogo documentado antes de su implementación en SQL.
